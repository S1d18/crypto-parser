# 📊 Supertrend Bot Dashboard

Современный веб-интерфейс для мониторинга торгового бота Supertrend с понятной статистикой для новичков.

## ✨ Особенности дизайна

### 🎨 Визуальные улучшения
1. **Glassmorphism эффекты** — полупрозрачные карточки с blur эффектом
2. **SVG иконки Lucide** — профессиональные векторные иконки вместо emoji
3. **Плавные transitions** — все hover эффекты с анимацией 0.3s
4. **Градиентные акценты** — яркие градиенты для важных метрик
5. **Визуальная иерархия** — shadows и borders для глубины
6. **Cursor-pointer** — на всех кликабельных элементах
7. **Анимированные индикаторы** — пульсирующий статус бота
8. **Улучшенная цветовая палитра** — современные финтех цвета
9. **Полностью responsive** — отлично выглядит на всех устройствах

### 📱 Адаптивность
- **Desktop** (1200px+): 4 карточки метрик в ряд
- **Tablet** (768-1199px): 2 карточки в ряд
- **Mobile** (<768px): 1 карточка, упрощенная таблица

## 🚀 Запуск

### Вариант 1: Напрямую
```bash
cd web
python run.py
```

### Вариант 2: Через модуль
```bash
python -m web.run
```

Откройте в браузере: **http://localhost:5001**

## 📊 Что отображается

### Панель метрик (верх)
- **Общая прибыль** — сумма всех PnL с градиентным текстом
- **Винрейт** — процент прибыльных сделок
- **Всего сделок** — количество + средний PnL
- **Лучшая сделка** — максимальный профит и худший убыток

### Активные позиции
- **4h Long стратегия** — текущая long позиция
- **10m Short стратегия** — текущая short позиция (хедж)

Для каждой позиции:
- Цена входа
- Текущая цена (TODO: добавить WebSocket)
- PnL в $ и %
- Stop Loss уровень
- Progress bar (до SL)
- Время открытия

### Таблица сделок
- Последние 50 сделок (фильтр: 20/50/100)
- Фильтр по таймфрейму (4h/10m)
- Время, тип, цены, PnL, причина закрытия
- Длительность сделки

## 🎯 Для новичков

### Цветовая семантика
- 🟢 **Зеленый** = Прибыль, позиция открыта, бот работает
- 🔴 **Красный** = Убыток, бот остановлен
- 🔵 **Синий** = Информация, кликабельные элементы
- 🟡 **Желтый** = Предупреждения

### Понятные состояния
- **"Нет открытой позиции"** — бот мониторит, но не торгует
- **Progress bar** — визуально показывает расстояние до Stop Loss
- **Badges** — "LONG ▲", "SHORT ▼", "Сигнал", "Stop Loss"
- **Animated status** — пульсирующая точка = бот работает

## 🔧 API Endpoints

### GET /api/status
Статус бота и текущие позиции
```json
{
  "bot_running": true,
  "last_update": "2026-02-10T12:00:00",
  "positions": [
    {
      "timeframe": "4h",
      "symbol": "BTCUSDT",
      "direction": "long",
      "entry_price": 43250.0,
      "sl_price": 41952.5
    }
  ]
}
```

### GET /api/statistics
Общая статистика по сделкам
```json
{
  "total_trades": 127,
  "profitable_trades": 83,
  "losing_trades": 44,
  "win_rate": 65.35,
  "total_pnl": 234.89,
  "best_trade": 45.60,
  "worst_trade": -28.30
}
```

### GET /api/trades?limit=50&timeframe=4h
История сделок
```json
[
  {
    "id": 1,
    "timeframe": "4h",
    "direction": "long",
    "entry_price": 43250.0,
    "close_price": 43890.0,
    "pnl": 25.60,
    "status": "closed",
    "close_reason": "signal"
  }
]
```

### GET /api/chart-data?days=30
Данные для графика баланса (кумулятивный PnL)
```json
[
  {
    "timestamp": "2026-02-10T12:00:00",
    "pnl": 234.89
  }
]
```

## 🎨 Кастомизация CSS

Основные CSS переменные в `dashboard.css`:

```css
:root {
    --bg-primary: #0a0e27;      /* Главный фон */
    --glass-bg: rgba(26, 31, 77, 0.4);  /* Glassmorphism */
    --green: #00e676;           /* Профит */
    --red: #ff5252;             /* Убыток */
    --blue: #448aff;            /* Акценты */
}
```

Градиенты:
- `--gradient-green` — для прибыли
- `--gradient-red` — для убытков
- `--gradient-blue` — для информации

## 🐛 Troubleshooting

### Порт 5001 занят
Измените порт в `app.py` и `run.py`:
```python
app.run(host="0.0.0.0", port=5002, debug=True)
```

### Нет данных
Убедитесь что `trades.db` существует в корневой директории:
```bash
ls -la ../trades.db
```

### Иконки не отображаются
Проверьте интернет-соединение (Lucide загружается с CDN).
Альтернатива: скачать lucide.js локально.
