# 🚀 Быстрый старт

## Запуск веб-дашборда

### Windows (самый простой способ):
```bash
# Дважды кликните на файл:
START_DASHBOARD.bat
```

### Альтернативный способ:
```bash
cd web
python run.py
```

Откройте в браузере: **http://localhost:5001**

---

## Что вы увидите

### ✅ Статус бота
Зеленая пульсирующая точка = бот работает

### 💰 4 карточки метрик
- Общая прибыль (с градиентом)
- Винрейт (% прибыльных сделок)
- Количество сделок
- Лучшая/худшая сделка

### 📊 Активные позиции
- **4h Long** — основная стратегия (слева)
- **10m Short** — хедж стратегия (справа)

Если позиция открыта, вы увидите:
- Цену входа
- Текущую цену
- PnL в $ и %
- Stop Loss уровень
- Progress bar (визуально показывает расстояние до SL)

### 📝 История сделок
Таблица с последними сделками:
- Фильтр по таймфрейму (4h/10m)
- Фильтр по количеству (20/50/100 сделок)
- Цветовое кодирование: зеленый = прибыль, красный = убыток

---

## Автообновление

Дашборд автоматически обновляется каждые 5 секунд:
- Статус бота
- Активные позиции
- Статистика

## Требования

- Python 3.11+
- Flask (уже в requirements.txt)
- Запущенный торговый бот (bot.py) или существующая БД trades.db

---

## Если возникли проблемы

### Порт 5001 занят
Измените порт в `web/app.py` (строка 145):
```python
app.run(host="0.0.0.0", port=5002, debug=True)
```

### База данных не найдена
Убедитесь что `trades.db` существует в корневой директории.
Если нет, запустите бота хотя бы один раз:
```bash
python bot.py
```

### Импорт ошибки
Активируйте виртуальное окружение:
```bash
.venv\Scripts\activate
```
